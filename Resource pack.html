The easiest way to make a Zenith resource pack is in Textured module using json scripts


The best IDE text editor is Visual Studio Code

Get VS Code

Attributions for each element

Showing description/info about the resource pack

              "description": "Text to be desplayed in the description"

Showing image preview. Any format of image is accepted(bmp, png, jpg)

              "previewImage": "img.png"

Texture loader attributions


Black key (Unpressed)

              "blackKey": "KeyBlack.png"

Black key (Pressed)

              "blackKeyPressed": "KeyBlackPressed.png"

Key White (Unpressed)

              "whiteKey": "WhiteKey.png"

Key white (Pressed)

              "whiteKeyPressed": "WhiteKeyPressed.png"

Bar or divider top

              "bar": "bar.png"

Notes texture loader

              "middleTexture": "note.png"

Note edges texrure loader

              "topTexture": "noteEdge.png"
            
              "bottomTexture": "noteEdge.png"

Other customizations


Same Width Notes

This attribution has 2 values: "true"(Same width notes is enabeld) or "false"(Same width notes is disabled)

Here is an preview about same width notes


When same width notes is "true" (Enabled) the notes and the keyboard align looks like in this image:


When Same Width notes is "false" (Disabled) the notes and keyboard align looks like this:

                "sameWidthNotes": "true"

Bar height attribution

               "barHeight": "20.24175824178824178"

Draken black keys notes attribute

                "darkenBlackNotes": "0.65"

Highlight notes when are falling onto the keyboard

                "highlightHitNotes": "0.5"

Notes and keyboard shaders

These attributes has 3 values: normal, hybrid, inverse


At notes seection

                "noteShader": "hybrid"
              
                "noteShader": "inverse"

At keyboard

Black keys

                "blackKeyShader": "inverse"
              
                "blackKeyShader": "hybrid"

White keys

                "whiteKeyShader": "inverse"
              
                "whiteKeyShader": "hybrid"

Black key offsets

                "blackKey3setOffset": 0.625
              
                  "blackKey2setOffse": 0.625

Is recommanded to use both attributions for better keyboard and notes re-align

Black keys width(Black Key Scale)

                "blackKeyScale": 0.481481481481481481481

Keyboard Height

                "keyboardHeight": 8.691358024691358024691358024691358024

Example script

            {
               //Any line that starts with "//" is a comment and can be removed
               "description": "My Zenith resource pack",
               "sameWidthNotes": "false",

               "previewImage": "preview.png",

               "blackKey": "keyBlack.png",
               "blackKeyPressed": "keyBlackPressed.png",
               "whiteKey": "whiteKey.png",
               "whiteKeyPressed": "whiteKeyPressed.png",
               "bar": "bar.png",

               "barHeight": 20.24175824178824178,

               "noteShader": "hybrid",
               "blackKeyShader": "inverse",
               "whiteKeyShader": "inverse",

               "blackKeyOversize": 3,
               "blackKeyPressedOversize": 1,
               "blackKey3setOffset": 0.625,
               "blackKey2setOffset": 0.625,
               "blackKeyScale": 0.481481481481481481481,
               "keyboardHeight": 8.691358024691358024691358024691358024,

               "blackKeysWhiteShade": "true",
               "blackKeyHeight": 40,
               //Note type maximum sizes (aspect ratios)
               //Warning: using more than one note height texture is much slower
               "notes":  [
                 {
                   //Size is relative to width. Size = note_height / note_width
                   "maxSize": 4,
                   //alwaysStretch means that it will never tile the texture
                    "alwaysStretch": "true",
                    "useEndCaps": "true",
                    "middleTexture": "note.png",
                    "topTexture": "noteEdge.png",
                    "bottomTexture": "noteEdge.png",
                    "topOversize": 0,
                    "bottomOversize": 0,
                    "darkenBlackNotes": 0.65,
                    "highlightHitNotes": 0.5
                  }
               ]
            }
